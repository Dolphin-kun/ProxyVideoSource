# 「動画プロキシ」動画読み込みプラグイン

### [最新バージョンをダウンロード](https://github.com/Dolphin-kun/ProxyVideoSource/releases/latest)

> [!Warning]
> ## __このプラグインはDLしただけで実行されます。__  
> 
> ffmpegが動くため、CPUを使います。  
> 設定をしていないとあるだけのCPUをごっそり使ってくるので気をつけてください。

## 概要
動画を読み込むときに軽量で低解像度の動画（プロキシ）を生成して、編集時にそれを使用して編集できるようになります。
ffmpegを使用します。一度も動画を出力したことがない場合使えないため、ご確認ください。

## 使い方
1. プラグイン導入後、動画を追加すると自動で使用されます。（メニューバー「ファイル」→「設定」にある「プラグイン」で『動画プロキシプラグイン』が読み込まれていればOKです。）

## パラメータ
設定画面の「動画読み込み」カテゴリに『動画プロキシ』があります

<img width="400" alt="image" src="https://github.com/user-attachments/assets/4b1a8eae-9f1a-430c-ac2f-edac5473f025" />

## 動作
動画を読み込む
↓
「最小ファイルサイズ」より大きければプロキシを生成
↓
プロキシ生成後、動画を再度読み込みする（一度動画アイテム以外のところにシークバーを持っていき、再度動画アイテムに乗せる）とプロキシが使われる

出力時は元の動画が使用されるようになっています。
